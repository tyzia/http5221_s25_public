Radio Button Example (Only One Can Be Selected)

Choose Your Favorite Fruit



Explanation:

In this group of radio buttons, only one option can be selected at a time because they all share the same name="fruit". If you select "Apple", the other two options ("Banana" and "Cherry") will be automatically deselected, and vice versa.

Checkbox Example (Multiple Selections Allowed)

Select Your Favorite Fruits

Explanation:

In this group of checkboxes, the user can select multiple options independently. For example, the user can select "Apple" and "Cherry" without deselecting "Banana", as each checkbox operates independently.
