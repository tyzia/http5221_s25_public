Task 1: Enable Overrides

  1. Go to Sources tab
  2. Open the Overrides tab (if not visible, see below)
  3. Select an empty folder on your computer
  4. Allow DevTools access when prompted
Don't see Overrides? Try:
  • Click the >> icon in Sources tab to find Overrides
  • Update Chrome to latest version
  • Alternatively, use Local Overrides in Chrome Flags

Task 2: Fix the CSS Path

  2. Change nonexistent-styles.css to styles.css
  3. Save (Ctrl+S/Cmd+S)
  4. Refresh page to see changes
